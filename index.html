Welcome to Nanapiyasa Learning Platform

This research presents an intelligent, multimodal platform designed to support students with Down syndrome through cognitive assessment, real-time engagement monitoring, and personalized learning recommendations using Artificial Intelligence and IoT technologies.

Domain Overview

Research Background, Literature Survey, and Methodology

This section explains the background of our research, including existing studies, identified gaps, and how our proposed system addresses real-world challenges.

Literature Survey

Modern technologies such as Artificial Intelligence (AI), Machine Learning (ML), and the Internet of Things (IoT) have significantly improved the way educational systems support diverse learners.

Research shows that gamified learning environments help students stay engaged while allowing systems to collect important cognitive data such as memory accuracy, reaction time, and error rates.

At the same time, IoT-based sensors can monitor physiological signals like heart rate, which provide insights into a student's attention and emotional state.

In addition, machine learning models are widely used to analyze learning patterns, classify student performance, and generate predictions. Educational dashboards also help teachers monitor progress and make informed decisions.

However, most existing solutions focus only on individual features rather than combining all these capabilities into one unified system.

Research Gap

Lack of Integrated Systems

Combining learning, monitoring, and analytics into one unified platform

Limited Solutions for Down Syndrome

Few systems specifically designed for students with Down syndrome

Minimal Real-time Adaptation

Insufficient use of real-time adaptive learning approaches

Lack of Vocational Guidance

Insufficient personalized vocational guidance systems

These limitations highlight the need for a more comprehensive and intelligent solution.

Research Problem

Students with Down syndrome often face difficulties in learning, communication, and social interaction. Traditional education systems are not flexible enough to support their individual needs.

There is a need for a system that can:

  • Understand each student's abilities
  • Monitor engagement in real time
  • Provide personalized learning support
  • Guide students toward suitable career paths

Key Problem: How can we design an intelligent system that uses both behavioral and physiological data to improve learning and support future independence?

Research Objectives

Key Objectives

Proposed Solution

Multimodal AI-Driven Learning Support System

This research introduces a multimodal AI-driven platform that integrates behavioral data, physiological signals, and machine learning to provide a comprehensive learning support system.

By analyzing both interaction-based and physiological data, the system provides real-time insights and personalized recommendations to educators and caregivers.

The Platform Combines

Gamified Cognitive Assessment

Interactive game-based assessments that evaluate cognitive abilities while maintaining student engagement through fun, adaptive challenges.

IoT-Based Physiological Monitoring

Real-time monitoring of physiological signals through IoT sensors to track stress levels, attention, and engagement during learning activities.

AI-Powered Analytics Dashboard

Comprehensive analytics platform that processes multimodal data to provide actionable insights and progress tracking for educators.

Vocational Recommendation System

Personalized career guidance system that matches individual strengths and abilities with suitable vocational pathways and opportunities.

Communication & Social Skills Training

Interactive modules designed to improve communication abilities and social interaction skills through structured exercises and real-world scenarios.

Data-Driven Approach

Structured Methodology for Intelligent Learning Support

The system follows a structured, data-driven approach consisting of five key stages:

Data Collection

Behavioral data such as memory accuracy and reaction time are collected through a gamified mobile application. Physiological data, including heart rate and variability, are captured using IoT devices.

Data Processing

Collected data undergoes noise filtering, normalization, and synchronization to ensure consistency and accuracy.

Feature Extraction

Key features are extracted from both behavioral and physiological datasets to represent cognitive performance and engagement levels.

Machine Learning

Supervised learning models are used to classify students into functional ability levels and predict engagement patterns.

Prediction & Output

The system generates insights, alerts, and recommendations based on analyzed data, supporting educators in decision-making.

System Architecture

Interconnected Components for Intelligent Learning Support

The system consists of multiple interconnected components working together to deliver intelligent learning support.

The architecture enables seamless data flow from user interaction to intelligent decision-making.

Data Collection Layer

Captures behavioral data from gamified applications and physiological data from IoT sensors.

Data Processing Layer

Cleans, normalizes, and synchronizes collected data for analysis and processing.

Machine Learning Layer

Performs classification and predictive analytics to generate insights and recommendations.

Application Layer

Provides dashboards, recommendations, and chatbot interaction for users and educators.

Results & Findings

Experimental Evaluation and Key Outcomes

The experimental evaluation demonstrates that integrating behavioral and physiological data significantly improves classification accuracy compared to single-data approaches.

The results confirm the effectiveness of a multimodal approach in enhancing learning analytics.

Key findings include

Improved Detection of Student Engagement Levels

Enhanced accuracy in identifying and monitoring student engagement patterns through multimodal data analysis.

Higher Accuracy in Functional Level Classification

Significant improvement in classifying students into appropriate functional ability levels using combined data sources.

Effective Early Identification of Attention Decline

Early detection systems successfully identify attention decline patterns, enabling timely intervention and support.

Meaningful Vocational Recommendations

Personalized career guidance aligned with individual student abilities and strengths, supporting successful vocational outcomes.
